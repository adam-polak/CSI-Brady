1. Navigate to FastAPI folder in terminal
2. Create venv
    run "python -m venv ./venv"
3. Activate venv
    run "source ./venv/bin/activate"
3. Install packages
    run "pip install -r ./requirements.txt"
4. Start FastAPI
    run "fastapi dev ./api.py"
5. When done
    run "deactivate" to exit venv
